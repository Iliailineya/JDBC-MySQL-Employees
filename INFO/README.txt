
Техстек: Maven, MySQL, JDBC, Lombok.


1) Визначаємо об'єкти (сутності) реального світу.
На основі цих об'єктів сформуємо таблиці БД та
моделі Java-класів

2) Складаємо SQL-запити SQLs.sql.

3) Створюємо Maven-проект.
В Advanced Settings можемо прописати org.example.app

4) Можемо змінити .gitignore.

5) Додаємо залежності.

6) Формуємо пакети, класи в org/example/app

7) Перевіряємо працездатність програми.

---------------------------------------------

УВАГА. При використанні бібліотеки Lombok,
можуть бути труднощі, оскільки вона не підтримується
в IntelliJ IDEA за замовчуванням.

Потрібно встановити плагін та налаштувати підтримку
анотацій.

Встановлюємо плагін:
Settings -> Plugins -> Marketplace -> шукаємо Lombok.
Після встановлення перезавантажуємо IntelliJ IDEA.

Включаємо обробку анотацій Lombok:
Settings -> Build, Execution, Deployment ->
Compiler -> Annotation Processors ->
Enable annotation processing.

---------------------------------------------

УВАГА. Можливо, після певних змін, буде необхідно
перезібрати проект.

Бічна панель Maven -> Lifececle -> clean ->
compile -> install
